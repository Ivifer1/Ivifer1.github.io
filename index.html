Sobre Mí

Ingeniero de datos en formación con enfoque en arquitecturas de datos prácticas. He construido tres pipelines completos que abarcan desde Data Warehouses en AWS hasta integraciones multi-plataforma, pasando por ETLs con Python, orquestación con Airflow/n8n y modelado dimensional con SQL avanzado.

Mi diferenciador es que no solo ejecuto scripts: diseño sistemas. Documento decisiones técnicas, escribo tests, contenerizo con Docker, despliego en la nube y pienso en la mantenibilidad a largo plazo.

RetailPulse Analytics

Data Warehouse dimensional en AWS que centraliza datos de ventas de 50 tiendas en 5 regiones. Modelado con Star Schema, 500K+ registros, KPIs analíticos con SQL avanzado (Window Functions, CTEs, Cohort Analysis).

Arquitectura AWS

Amazon S3 Raw CSVs
EC2 t2.micro Ubuntu 22.04
PostgreSQL 18 Star Schema
psql Local Analytics
AWS Free Tier — EC2 + S3 + IAM + PostgreSQL DW
Star Schema: 5 dimensiones + 1 fact table
500K registros de ventas sintéticos
Window Functions: LAG, RANK, PARTITION
Cohort Analysis de retención
KPIs: LTV, MoM Growth, Product Rotation
Costo $0 en AWS Free Tier

Pipeline ETL completo que extrae datos de la Fake Store API, los transforma con Pandas y los carga en PostgreSQL. Orquestado con Apache Airflow y visualizado en Metabase. Todo contenerizado con Docker Compose.
